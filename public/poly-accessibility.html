Accessibility Statement

Last Updated: January 27, 2026

Our Commitment

Poly People Printing is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

This website is designed to conform to WCAG 2.1 Level AA standards. The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities.

Our website meets the following standards:

  • Americans with Disabilities Act (ADA) Title III
  • Section 508 of the Rehabilitation Act
  • WCAG 2.1 Level AA

Accessibility Features

Our website includes the following accessibility features:

  • Alternative Text: All images include descriptive alt text for screen readers
  • Keyboard Navigation: Full site functionality available via keyboard
  • Color Contrast: Text meets minimum 4.5:1 contrast ratio
  • Responsive Design: Works on all devices and screen sizes
  • Skip Navigation: Skip links allow quick access to main content
  • Form Labels: All form fields have clear, associated labels
  • ARIA Landmarks: Proper semantic structure for assistive technologies
  • Focus Indicators: Clear visual indicators for keyboard focus

Compatible Technologies

This website is designed to work with the following assistive technologies:

  • Screen readers (JAWS, NVDA, VoiceOver, TalkBack)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation
  • Switch devices and alternative input methods

Known Limitations

Despite our best efforts, some limitations may exist:

  • Third-party content or embedded elements may have their own accessibility limitations
  • User-uploaded content may not meet accessibility standards
  • Some animations can be paused or stopped using browser settings

Technical Specifications

Website accessibility relies on the following technologies:

  • HTML5
  • CSS3
  • JavaScript (with graceful degradation)
  • WAI-ARIA

Assessment Approach

Poly People Printing assessed the accessibility of this website using the following methods:

  • Self-evaluation and testing
  • Automated accessibility testing tools
  • Manual testing with assistive technologies
  • Keyboard-only navigation testing

Continuous Improvement

We are committed to maintaining and improving the accessibility of our website. This includes:

  • Regular accessibility audits
  • Staff training on accessibility best practices
  • Monitoring new WCAG guidelines and standards
  • Incorporating accessibility into our design and development process
  • Responding promptly to accessibility feedback
